## 📌 مميزات المرجع (Features)
* **شرح باللغة العربية:** التعليقات داخل الكود تشرح وظيفة كل دالة بأسلوب مباشر ومفهوم.
* **تغطية شاملة للأساسيات:** يتضمن إنشاء المصفوفات، العمليات الإحصائية، والدمج (Joining).
* **تطبيقات متقدمة للـ ML:** شرح وتطبيق عملي لكيفية دمج ميزة الـ `Vectorization` مع مصفوفة الوحدة `np.eye()` لتوليد الـ `One-Hot Encoding` فائق السرعة بدون استخدام حلقات التكرار التقليدية (`for loops`).

---

## 🛠️ الأقسام الرئيسية في الكود (Main Sections)
1. **Array Creation:** إنشاء المصفوفات أحادية وثنائية الأبعاد والخصائص (`shape`, `ndim`, `size`).
2. **Special Arrays & Reshaping:** الأصفار، المصفوفات الأحادية، مصفوفة الوحدة، والـ `reshape` والـ `flatten`.
3. **Random Generation:** توليد الأرقام العشوائية والتوزيع الطبيعي.
4. **Operations & Broadcasting:** العمليات الحسابية والدوال الإحصائية (`mean`, `sum`, `max`).
5. **View vs Copy:** الفرق الجوهري في إدارة الذاكرة بين الرؤية والنسخة المستقلة.
6. **Advanced Indexing:** تطبيق الـ `One-Hot Encoding` للميزات الفئوية (Categorical Features).
